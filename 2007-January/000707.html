On 1/15/07, Ryan Flegel <rflegel at gmail.com> wrote:
>
> Hi there,
>
> For those who haven't already noticed, the SuperTux project moved to a
> new host over the weekend. The reason for this was the poor
> performance from BerliOS (the result of a DDoS attack). In particular,
> the website has been very non-responsive, but lately it has been
> affecting more important things like the Subversion repository. In
> addition, moving the website to this new host gives us more freedom in
> what kind of applications/services we can run on it.
>
> We have moved the website, Subversion repository and the mailing lists
> to the new host.
>
> These are the new addresses:
>
> Website: http://supertux.lethargik.org/
> Anonymous Subversion: http://supertux.lethargik.org/svn/supertux
> Mailing lists: supertux-devel at lists.lethargik.org,
> supertux-commit at lists.lethargik.org
>
> As an extra bonus, we have set up the Mantis bug tracker on the new
> website. It can be found here: http://supertux.lethargik.org/bugs/
>
> If you are a developer and require Subversion access, please file a
> ticket in the bug tracker under the "meta" project.
>
> If you were subscribed to the old BerliOS mailing lists (like this
> one) you have been automatically subscribed to the new ones and should
> already be receiving email from there. If you had an account on the
> wiki at BerliOS, the same account will exist on the new host. However,
> you might want to take this time to sign up for the new bug tracker.
>
> --
> Ryan Flegel
