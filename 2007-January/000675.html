The warning message you receive is non-fatal. The author of autoopts.m4 
just didn't have the brains to put AG_PATH_AUTTOOPTS into square 
brackets. (Or the m4 language is stupid. I didn't create packages using 
autoconf so I don't know.)

As for jam's error messages: right after the first ones appear, press 
Ctrl+C. (This terminates jam.) Then, copy the first twenty lines of 
error messages to pastebin and link it over here. GCC is known for 
trying to work around unknown language elements by just ignoring them, 
but in most cases, this just generates hundreds of superfluous syntax 
errors because the unknown element was a type definition or whatnot.

~~ Ondra
