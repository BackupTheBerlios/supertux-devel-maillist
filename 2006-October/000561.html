> I'd like to use the SuperTux music for background music because the game is based on Supertux. However, Game Maker cannot understand .mod files, so I can't use them. Any suggestions?

SuperTux Milestone 2 will have Vorbis music. You can find drafts of the 
music in the SVN repository 
<svn://svn.berlios.de/supertux/trunk/supertux>, for example 
<http://svn.berlios.de/viewcvs/*checkout*/supertux/trunk/supertux/data/music/chipdisko.ogg?rev=3335>.


Note, however, that most data files of SuperTux are *not* in the public 
domain, but released under a strict license, which you will have to 
adhere to if you use them.
