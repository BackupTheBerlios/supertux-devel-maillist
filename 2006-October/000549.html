Krishnamurti L. L. V. Nunes wrote:
> OpenAL `make install' gives me the following:
> 
> ----------------------------------------------------------------------
> Libraries have been installed in:
>    /usr/local/lib
> [...] 
>    - add LIBDIR to the `LD_LIBRARY_PATH' environment variable
>      during execution

This sounds like a plan. Try running the following commands in bash:

export LD_LIBRARY_PATH=$LD_LIBRARY_PATH:/usr/local/lib
./supertux


If ./configure took care of adding the correct -L switches this should 
work. If it does you can still try to make this more permanent (e.g. by 
adding to your .bashrc, moving the libraries or re-configuring the 
run-time linker)
