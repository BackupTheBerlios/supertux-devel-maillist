Thanks for asking. Actually there's quite a lot to do besides improving 
the engine. Just have a look at "Can I help with Milestone 2?" on 
<http://supertux.berlios.de/wiki/index.php/FAQ> for some good starting 
points.


If, for example, you want to help translating SuperTux, see 
<http://supertux.berlios.de/wiki/index.php/Translation> for a quick 
summary on how to help out there.

For inspiration on what graphics the game needs most, see 
<http://supertux.berlios.de/wiki/index.php/TODO#Graphics>.

We also need some SFX - read 
<http://supertux.berlios.de/wiki/index.php/TODO#Sounds> for a list of 
missing sound effects.


If you need any help setting up tools for SuperTux development, you'll 
find people involved with SuperTux on the SuperTux IRC channel 
<http://supertux.berlios.de/wiki/index.php/IRC_Channel>.
