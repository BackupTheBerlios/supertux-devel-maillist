> I'm assuming that you're using the development version from SVN since
> you've been on the list a while.  There is an internal variable called
> "LOGICAL_FPS" which is defined in two places, src/game_session.cpp and
> src/mainloop.cpp.  It's currently set to 64.  You should be able to
> change this value to something else and recompile.  It says that it's
> supposed to be a power of 2, but I'm not sure if that's really
> required.  You can set the value to something high like 1024.  I get
> around 300-350fps by doing this.  Some animations are a little bit
> messed up and the keyboard responses don't always work 100%, but it's
> still somewhat playable.
>
> If you raise these values and it's still pegged at 60fps, then it could
> be your video card driver.
