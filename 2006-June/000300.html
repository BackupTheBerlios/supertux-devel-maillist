In case you didn't know: The e-mail address you wrote to earlier is a

Julian Marchant wrote:
> There are three things I cannot do in the built-in level editor:
>   1.I cannot make boxes breakable
>   2.I cannot use backscroll or autoscroll
>   3.I cannot put in a reset point
>   I would REALLY like to be able to do all three of these things.

There are indeed some features like breakable boxes and reset points 
that either didn't make it to the level editor in time for the release 
or were simply forgotten to place in. Most of these features can be used 
with Flexlay, which can make SuperTux 0.1.x levels.

Other features (like backscrolling) are in the engine, but unused 
because they are incomplete and/or broken. If you want to make stable, 
good-looking levels you shouldn't use these, but you can.

The SuperTux fansite forums have quite a lot of posts explaining how to 
do stuff that's impossible to do using the built-in level editor. For 
example, see <http://www.supertux.info/forums/viewtopic.php?id=194> for 
an explanation concerning autoscrolling and breakable wooden boxes.

Rest assured that all three of the features mentioned will be accessible 
in the SuperTux 0.2.x level editor.


Good luck with your levels - and don't forget to submit good ones, e.g. 
at <http://www.supertux.info/upload.html>.
