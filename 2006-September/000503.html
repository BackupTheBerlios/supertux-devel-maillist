Well this year I had a course at university about operating system
design and implementation with a practical part where we had to
implement a toy operating system. Well our group was thinking that an
operating system without games is boring so we ported SDL and supertux
on it ;-) So now we have yet another platform that runs supertux 0.1.3.

You can find a few more informations and the source in Timos blog entry:
   http://idlebox.net/2006/20060914-SDIOS06.blog
or simply download a ready to run vmware image here:
   http://idlebox.net/2006/SDIOS06/sdios06-vmware-image-v1.0.tar.bz2

More information about the OS can be found in the README file:
   http://idlebox.net/2006/SDIOS06/README.source
